## Project Overview
An interactive Power BI dashboard created to visualize and analyze sales trends, key performance indicators (KPIs), and customer insights from raw datasets.

## Key Insights & Features
* **Sales Trends:** Visualizing revenue growth over time.
* **Product Performance:** Identifying top-selling products and categories.
* **Regional Analysis:** Mapping sales distribution across different locations.
* **Interactive Filters:** Allows users to filter data by date, region, and product type.

## Tools Used
* **Power BI Desktop:** For data modeling and visualization.
* **Excel/CSV:** As the primary data source.
* **DAX (Data Analysis Expressions):** Created custom measures for total sales and profit margins.

## How to View
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
